# 按钮回调参数说明

## 回调函数签名

按钮点击回调函数有两个参数：

```易语言
.子程序 按钮点击回调, , , stdcall
.参数 按钮ID, 整数型
.参数 父窗口句柄, 整数型
```

## 参数说明

### 参数1：按钮ID（整数型）

- **含义**：被点击按钮的唯一标识符
- **用途**：用于区分是哪个按钮被点击
- **获取方式**：创建按钮时的返回值

```易语言
按钮1 = 创建Emoji按钮_辅助(窗口句柄, "", "确定", 10, 10, 100, 30, #COLOR_PRIMARY)
' 按钮1 变量中存储的就是按钮ID
```

### 参数2：父窗口句柄（整数型）

- **含义**：按钮所在窗口的句柄
- **用途**：用于识别按钮属于哪个窗口
- **特殊情况**：
  - 如果按钮在主窗口上，这个参数就是主窗口句柄
  - 如果按钮在分组框中，这个参数仍然是主窗口句柄（不是分组框句柄）

## 使用示例

### 基本用法

```易语言
.程序集变量 窗口句柄, 整数型
.程序集变量 按钮1, 整数型
.程序集变量 按钮2, 整数型

.子程序 _启动子程序, 整数型

窗口句柄 = 创建Emoji窗口("测试", 400, 300)
按钮1 = 创建Emoji按钮_辅助(窗口句柄, "", "确定", 10, 10, 100, 30, #COLOR_PRIMARY)
按钮2 = 创建Emoji按钮_辅助(窗口句柄, "", "取消", 120, 10, 100, 30, #COLOR_DANGER)

设置按钮点击回调(&按钮点击回调)
运行消息循环()

返回(0)

.子程序 按钮点击回调, , , stdcall
.参数 按钮ID, 整数型
.参数 父窗口句柄, 整数型

.如果真(按钮ID = 按钮1)
    调试输出("点击了确定按钮")
.如果真结束

.如果真(按钮ID = 按钮2)
    调试输出("点击了取消按钮")
.如果真结束
```

### 分组框中的按钮

```易语言
.程序集变量 窗口句柄, 整数型
.程序集变量 分组框1, 整数型
.程序集变量 按钮1, 整数型  ' 主窗口上的按钮
.程序集变量 按钮2, 整数型  ' 分组框中的按钮

.子程序 _启动子程序, 整数型

窗口句柄 = 创建Emoji窗口("测试", 400, 300)

' 在主窗口上创建按钮
按钮1 = 创建Emoji按钮_辅助(窗口句柄, "", "主窗口按钮", 10, 10, 120, 30, #COLOR_PRIMARY)

' 创建分组框
分组框1 = 创建分组框_辅助(窗口句柄, 10, 50, 200, 100, "分组框", #COLOR_BORDER_BASE, #白色)

' 在分组框中创建按钮
按钮2 = 创建Emoji按钮_辅助(分组框1, 10, 10, 120, 30, "", "分组框按钮", #COLOR_SUCCESS)

设置按钮点击回调(&按钮点击回调)
运行消息循环()

返回(0)

.子程序 按钮点击回调, , , stdcall
.参数 按钮ID, 整数型
.参数 父窗口句柄, 整数型

调试输出("按钮ID=" + 到文本(按钮ID) + ", 父窗口句柄=" + 到文本(父窗口句柄))

.如果真(按钮ID = 按钮1)
    调试输出("点击了主窗口上的按钮")
    ' 父窗口句柄 = 窗口句柄
.如果真结束

.如果真(按钮ID = 按钮2)
    调试输出("点击了分组框中的按钮")
    ' 父窗口句柄 = 窗口句柄（注意：不是分组框句柄）
.如果真结束
```

## 重要注意事项

### 1. 父窗口句柄的含义

⚠️ **重要**：即使按钮在分组框中创建，`父窗口句柄` 参数仍然是主窗口句柄，而不是分组框句柄。

```易语言
' 错误理解 ❌
.如果真(父窗口句柄 = 分组框1)  ' 这永远不会成立！
    ' ...
.如果真结束

' 正确理解 ✅
.如果真(父窗口句柄 = 窗口句柄)  ' 这才是正确的
    ' ...
.如果真结束
```

### 2. 按钮ID的唯一性

- 每个按钮的ID在同一个窗口中是唯一的
- 按钮ID从1000开始递增
- 不同窗口的按钮可能有相同的ID

```易语言
' 窗口1
窗口1 = 创建Emoji窗口("窗口1", 400, 300)
按钮1 = 创建Emoji按钮_辅助(窗口1, "", "按钮", 10, 10, 100, 30, #COLOR_PRIMARY)
' 按钮1 的ID可能是 1000

' 窗口2
窗口2 = 创建Emoji窗口("窗口2", 400, 300)
按钮2 = 创建Emoji按钮_辅助(窗口2, "", "按钮", 10, 10, 100, 30, #COLOR_PRIMARY)
' 按钮2 的ID也可能是 1000（因为是不同窗口）

' 所以需要同时判断按钮ID和父窗口句柄
.如果真(按钮ID = 按钮1 且 父窗口句柄 = 窗口1)
    调试输出("窗口1的按钮被点击")
.如果真结束
```

### 3. 全局回调

按钮点击回调是全局的，所有窗口的所有按钮共享同一个回调函数。

```易语言
' 只需要设置一次
设置按钮点击回调(&按钮点击回调)

' 这个回调会处理所有窗口的所有按钮点击
.子程序 按钮点击回调, , , stdcall
.参数 按钮ID, 整数型
.参数 父窗口句柄, 整数型

' 处理窗口1的按钮
.如果真(父窗口句柄 = 窗口1)
    .如果真(按钮ID = 窗口1按钮1)
        ' ...
    .如果真结束
.如果真结束

' 处理窗口2的按钮
.如果真(父窗口句柄 = 窗口2)
    .如果真(按钮ID = 窗口2按钮1)
        ' ...
    .如果真结束
.如果真结束
```

## 最佳实践

### 1. 使用有意义的变量名

```易语言
' 不好的命名 ❌
.程序集变量 按钮1, 整数型
.程序集变量 按钮2, 整数型
.程序集变量 按钮3, 整数型

' 好的命名 ✅
.程序集变量 确定按钮, 整数型
.程序集变量 取消按钮, 整数型
.程序集变量 全选按钮, 整数型
```

### 2. 添加调试输出

在开发阶段，建议添加调试输出以便排查问题：

```易语言
.子程序 按钮点击回调, , , stdcall
.参数 按钮ID, 整数型
.参数 父窗口句柄, 整数型

' 调试输出
调试输出("按钮点击 - ID:" + 到文本(按钮ID) + ", 窗口:" + 到文本(父窗口句柄))

' 业务逻辑
.如果真(按钮ID = 确定按钮)
    调试输出("执行确定操作")
    ' ...
.如果真结束
```

### 3. 分组处理按钮

对于复杂的界面，可以按功能分组处理按钮：

```易语言
.子程序 按钮点击回调, , , stdcall
.参数 按钮ID, 整数型
.参数 父窗口句柄, 整数型

' 处理分组框1的按钮
.如果真(按钮ID = 分组框1清除按钮 或 按钮ID = 分组框1全选按钮)
    处理分组框1按钮(按钮ID)
    返回()
.如果真结束

' 处理分组框2的按钮
.如果真(按钮ID = 分组框2清除按钮 或 按钮ID = 分组框2全选按钮)
    处理分组框2按钮(按钮ID)
    返回()
.如果真结束

' 处理主窗口按钮
处理主窗口按钮(按钮ID)

.子程序 处理分组框1按钮
.参数 按钮ID, 整数型

.如果真(按钮ID = 分组框1清除按钮)
    ' ...
.如果真结束

.如果真(按钮ID = 分组框1全选按钮)
    ' ...
.如果真结束
```

## 常见问题

### Q1: 为什么分组框中的按钮，父窗口句柄不是分组框句柄？

A: 因为按钮实际上是绘制在主窗口上的，分组框只是一个视觉容器。按钮的坐标会自动转换为主窗口坐标，所以父窗口句柄始终是主窗口。

### Q2: 如何判断按钮是否在分组框中？

A: 无法直接从回调参数判断。建议使用命名约定或额外的标记变量：

```易语言
' 方法1：命名约定
.程序集变量 分组框1_全选按钮, 整数型
.程序集变量 分组框2_清除按钮, 整数型

' 方法2：使用标记
.程序集变量 分组框1按钮列表, 整数型, , "0"

分组框1_全选按钮 = 创建Emoji按钮_辅助(分组框1, ...)
加入成员(分组框1按钮列表, 分组框1_全选按钮)

' 在回调中判断
.如果真(是否为成员(分组框1按钮列表, 按钮ID))
    调试输出("这是分组框1中的按钮")
.如果真结束
```

### Q3: 可以为不同按钮设置不同的回调吗？

A: 不可以。按钮点击回调是全局的，所有按钮共享同一个回调函数。需要在回调函数内部通过按钮ID来区分不同的按钮。
