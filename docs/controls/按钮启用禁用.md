# 按钮启用/禁用功能

## 概述

按钮控件支持启用和禁用状态。禁用的按钮会显示为灰色,无法点击,也不会触发悬停效果。

## DLL 命令

### 1. 启用按钮

```
.DLL命令 启用按钮, , "emoji_window.dll", "EnableButton"
    .参数 父窗口句柄, 整数型, , 按钮所在的父窗口句柄
    .参数 按钮ID, 整数型, , 按钮ID
    .参数 启用, 逻辑型, , 真=启用，假=禁用
```

### 2. 禁用按钮

```
.DLL命令 禁用按钮, , "emoji_window.dll", "DisableButton"
    .参数 父窗口句柄, 整数型, , 按钮所在的父窗口句柄
    .参数 按钮ID, 整数型, , 按钮ID
```

`禁用按钮` 是 `启用按钮` 的快捷方式,等同于 `启用按钮(父窗口句柄, 按钮ID, 假)`

## 使用示例

### 基本用法

```
.版本 2

.程序集 窗口程序集1
.程序集变量 窗口1, 整数型
.程序集变量 按钮1, 整数型
.程序集变量 按钮2, 整数型
.程序集变量 按钮3, 整数型

.子程序 _启动子程序, 整数型

' 创建三个按钮

按钮2 = 创建Emoji按钮_辅助 (窗口1, "🔒", "禁用按钮2", 50, 110, 120, 40, #COLOR_DANGER)
按钮3 = 创建Emoji按钮_辅助 (窗口1, "🔓", "启用按钮2", 50, 170, 120, 40, #COLOR_SUCCESS)

' 设置按钮回调
设置按钮点击回调 (&按钮回调)

' 初始状态：按钮2 被禁用

.子程序 按钮回调, , , stdcall
.参数 按钮ID, 整数型
.参数 父窗口句柄, 整数型

.如果真 (按钮ID = 按钮1)
    信息框 ("你点击了正常按钮", 0, "提示")
.如果真结束

.如果真 (按钮ID = 按钮2)
    ' 这个按钮被禁用时不会触发回调

.如果真 (按钮ID = 按钮3)
    ' 切换按钮2的启用状态
```

### 动态切换示例

```
.子程序 按钮回调, , , stdcall
.参数 按钮ID, 整数型
.参数 父窗口句柄, 整数型
.局部变量 按钮2已启用, 逻辑型, 静态

' 初始状态为禁用
按钮2已启用 = 假

.如果真 (按钮ID = 切换按钮)
    .如果真 (按钮2已启用)
        ' 当前是启用状态,切换为禁用
        禁用按钮 (父窗口句柄, 按钮2)
        按钮2已启用 = 假
        信息框 ("已禁用按钮2", 0, "")
    .否则
        ' 当前是禁用状态,切换为启用
        启用按钮 (父窗口句柄, 按钮2, 真)
        按钮2已启用 = 真
        信息框 ("已启用按钮2", 0, "")
    .如果真结束
.如果真结束
```

## 视觉效果

### 启用状态
- 正常的按钮颜色
- 鼠标悬停时有高亮效果
- 可以点击并触发回调

### 禁用状态
- 浅灰色背景 (#F5F7FA)
- 浅灰色边框 (#E4E7ED)
- 浅灰色文本 (#C0C4CC)
- 鼠标悬停无效果
- 点击无响应

## 与分组框的配合

当分组框被禁用时,分组框内的所有按钮也会自动禁用:

```
' 禁用分组框会同时禁用分组框内的所有按钮
启用分组框 (分组框1, 假)

' 启用分组框会同时启用分组框内的所有按钮
启用分组框 (分组框1, 真)
```

## 注意事项

1. **父窗口句柄**: 必须传入按钮所在的父窗口句柄,不是按钮句柄
2. **按钮ID**: 使用创建按钮时返回的按钮ID
3. **禁用状态**: 禁用的按钮不会触发任何回调函数
4. **分组框联动**: 分组框的启用/禁用会自动影响其内部的所有按钮
5. **立即生效**: 调用后会立即重绘按钮,无需手动刷新

## 常见应用场景

1. **表单验证**: 输入不完整时禁用提交按钮
2. **操作限制**: 某些条件不满足时禁用特定操作按钮
3. **状态切换**: 根据应用状态动态启用/禁用按钮
4. **权限控制**: 根据用户权限启用/禁用功能按钮
5. **流程控制**: 按步骤启用/禁用下一步按钮
