# 分组框快速参考卡片

## 🎯 一句话总结

**分组框现在可以直接作为父窗口，子控件坐标相对于分组框内容区域，无需手动添加！**

---

## 📝 基本用法

```易语言
' 1. 创建分组框
分组框1 = 创建分组框_辅助(窗口句柄, 20, 20, 260, 150, "标题", #COLOR_BORDER_BASE, #白色)

' 2. 直接在分组框中创建子控件（坐标相对于分组框）
单选按钮1 = 创建单选按钮_辅助(分组框1, 10, 10, 100, 30, "选项1", 1, 真, #黑色, #白色)
复选框1 = 创建复选框_辅助(分组框1, 10, 50, 100, 30, "选项2", 假, #黑色, #白色)

' 3. 完成！无需其他操作
```

---

## 🎨 坐标系统

```
分组框 (x, y, width, height)
┌─────────────────────────────┐
│  标题文本 (高度约25px)       │
├─────────────────────────────┤
│ ← 10px →                    │
│    ↓ (0,0) 坐标原点         │
│    子控件从这里开始          │
│                             │
│                             │
└─────────────────────────────┘
```

- **原点位置**：分组框内容区域左上角
- **左边距**：自动添加 10px
- **顶部边距**：自动添加 25px（标题高度）

## 🔄 新旧对比

### ❌ 旧方式（不推荐）
```易语言
分组框1 = 创建分组框_辅助(窗口句柄, 20, 20, 260, 150, "标题", ...)
按钮1 = 创建按钮_辅助(窗口句柄, 40, 60, 100, 30, "按钮", ...)  ' 窗口坐标
添加子控件到分组框(分组框1, 按钮1)  ' 手动添加
```

### ✅ 新方式（推荐）
```易语言
分组框1 = 创建分组框_辅助(窗口句柄, 20, 20, 260, 150, "标题", ...)
按钮1 = 创建按钮_辅助(分组框1, 10, 10, 100, 30, "按钮", ...)  ' 分组框坐标
' 自动添加，无需手动调用
```

## 💡 常见场景

### 场景1：单选按钮组
```易语言
分组框1 = 创建分组框_辅助(窗口句柄, 20, 20, 200, 120, "性别", ...)
单选按钮1 = 创建单选按钮_辅助(分组框1, 10, 10, 80, 30, "男", 1, 真, ...)
单选按钮2 = 创建单选按钮_辅助(分组框1, 10, 50, 80, 30, "女", 1, 假, ...)
```

### 场景2：复选框组
```易语言
分组框2 = 创建分组框_辅助(窗口句柄, 240, 20, 200, 160, "兴趣", ...)
复选框1 = 创建复选框_辅助(分组框2, 10, 10, 100, 30, "编程", 假, ...)
复选框2 = 创建复选框_辅助(分组框2, 10, 50, 100, 30, "音乐", 假, ...)
复选框3 = 创建复选框_辅助(分组框2, 10, 90, 100, 30, "运动", 假, ...)
```

### 场景3：混合控件（包含按钮）
```易语言
分组框3 = 创建分组框_辅助(窗口句柄, 20, 160, 420, 200, "设置", ...)
标签1 = 创建标签_辅助(分组框3, 10, 10, 80, 30, "用户名:", ...)
编辑框1 = 创建编辑框_辅助(分组框3, 100, 10, 200, 30, ...)
复选框1 = 创建复选框_辅助(分组框3, 10, 50, 150, 30, "记住密码", ...)
```

```易语言

分组框1 = 创建分组框_辅助(窗口句柄, 20, 20, 260, 150, "选项", ...)
复选框1 = 创建复选框_辅助(分组框1, 10, 10, 100, 30, "选项1", ...)
复选框2 = 创建复选框_辅助(分组框1, 10, 50, 100, 30, "选项2", ...)
```

---

## ⚠️ 注意事项

1. **坐标原点**：(0,0) 对应分组框内容区域左上角，不是分组框本身的左上角
2. **自动边距**：系统会自动添加边距，无需手动计算
3. **兼容性**：旧代码仍然有效，可以继续使用
4. **不要混用**：同一个分组框不要混用新旧两种方式

## 🐛 常见错误

### 错误1：坐标计算错误
```易语言
❌ 错误：
分组框1 = 创建分组框_辅助(窗口句柄, 20, 20, ...)
按钮1 = 创建按钮_辅助(分组框1, 40, 60, ...)  ' 错误！这是窗口坐标

✅ 正确：
分组框1 = 创建分组框_辅助(窗口句柄, 20, 20, ...)
按钮1 = 创建按钮_辅助(分组框1, 10, 10, ...)  ' 正确！这是分组框相对坐标
```

### 错误2：仍然手动添加
```易语言
❌ 错误：
按钮1 = 创建按钮_辅助(分组框1, 10, 10, ...)
添加子控件到分组框(分组框1, 按钮1)  ' 不需要！会重复添加

✅ 正确：
按钮1 = 创建按钮_辅助(分组框1, 10, 10, ...)
' 无需手动添加
```
