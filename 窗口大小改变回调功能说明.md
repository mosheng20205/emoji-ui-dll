## 🎯 使用场景

1. **响应式布局**
   - 窗口大小改变时自动调整子控件位置和大小
   - 保持控件之间的相对位置关系

2. **TabControl 自适应**
   - 窗口放大时，TabControl 自动扩展
   - 窗口缩小时，TabControl 自动收缩

3. **编辑框和标签自适应**
   - 根据窗口宽度调整编辑框宽度
   - 保持标签和编辑框的对齐关系

4. **动态布局调整**
   - 根据窗口大小重新计算控件布局
   - 实现类似 CSS Flexbox 的自适应效果

## ⚠️ 注意事项

1. **回调函数必须使用 `__stdcall` 调用约定**
   - 易语言子程序默认就是 stdcall
   - 参数顺序：窗口句柄、宽度、高度

2. **在回调中调整控件大小**
   - 使用 `MoveWindow` API 调整 TabControl
   - 使用 `设置编辑框位置` 调整编辑框
   - 使用 `设置标签位置` 调整标签

3. **避免在回调中执行耗时操作**
   - 窗口大小改变时会频繁触发回调
   - 只做必要的控件位置调整
   - 避免复杂的计算和 I/O 操作

## 🔍 技术细节

### WM_SIZE 消息处理

```cpp
case WM_SIZE: {
    if (state) {
        UINT width = LOWORD(lparam);
        UINT height = HIWORD(lparam);
        state->render_target->Resize(D2D1::SizeU(width, height));
        
        // 调用窗口大小改变回调
        if (g_window_resize_callback) {
            g_window_resize_callback(hwnd, (int)width, (int)height);
        }
    }
    return 0;
}
```

### 回调函数签名

- **C++**: `void __stdcall WindowResizeCallback(HWND hwnd, int width, int height)`
- **易语言**: `子程序 _窗口大小改变(窗口句柄: 整数型, 新宽度: 整数型, 新高度: 整数型)`
