# C# 示例项目说明

## 📋 项目概述

这是一个完整的 C# 示例项目，展示如何使用 P/Invoke 调用 emoji_window.dll，实现带有 Unicode 表情的 Windows 桌面应用程序。

## 🎯 设计目标

1. **完整性**：覆盖主要控件（窗口、标签、按钮、分组框、复选框、信息框）
2. **可运行**：开箱即用，无编译错误
3. **双平台**：同时支持 32位 和 64位
4. **参考价值**：代码结构清晰，注释详细，易于学习

## 🔧 技术要点

### 1. P/Invoke 声明

```csharp
[DllImport("emoji_window.dll", CallingConvention = CallingConvention.StdCall)]
public static extern IntPtr create_window_bytes(
    byte[] titleBytes,
    int titleLen,
    int width,
    int height
);
```

**关键点：**
- 使用 `StdCall` 调用约定
- 字符串参数使用 `byte[]` + `int` 传递（UTF-8 编码）
- 返回值使用 `IntPtr` 表示句柄

### 2. UTF-8 编码转换

```csharp
public static byte[] StringToUtf8Bytes(string str)
{
    if (string.IsNullOrEmpty(str))
        return new byte[0];
    return Encoding.UTF8.GetBytes(str);
}
```

**原因：**
- DLL 使用 UTF-8 编码
- C# 默认使用 UTF-16 编码
- 必须手动转换

### 3. 回调函数管理

```csharp
// ✅ 正确：使用静态字段保持引用
private static EmojiWindowNative.ButtonClickCallback buttonCallback;

buttonCallback = OnButtonClick;
EmojiWindowNative.set_button_click_callback(buttonCallback);
```

**注意：**
- 必须保持委托引用，防止被垃圾回收
- 使用静态字段或实例字段
- 不能使用局部变量

### 4. 颜色值转换

```csharp
public static uint ARGB(byte a, byte r, byte g, byte b)
{
    return (uint)((a << 24) | (r << 16) | (g << 8) | b);
}
```

**格式：**
- ARGB 格式：`0xAARRGGBB`
- Alpha 通道：255 = 不透明，0 = 完全透明

## ⚠️ 常见陷阱

### 1. 回调被回收

```csharp
// ❌ 错误
void SetupCallback()
{
    var callback = new ButtonClickCallback(OnButtonClick);
    EmojiWindowNative.set_button_click_callback(callback);
    // callback 离开作用域后会被回收！
}

// ✅ 正确
private static ButtonClickCallback buttonCallback;
void SetupCallback()
{
    buttonCallback = OnButtonClick;
    EmojiWindowNative.set_button_click_callback(buttonCallback);
}
```

### 2. 平台不匹配

```csharp
// ❌ 错误：AnyCPU 会导致加载错误的 DLL
<PlatformTarget>AnyCPU</PlatformTarget>

// ✅ 正确：明确指定平台
<PlatformTarget>x86</PlatformTarget>  // 或 x64
```

### 3. 字符串编码

```csharp
// ❌ 错误：直接传递字符串
CreateLabel(hwnd, "Hello", ...);

// ✅ 正确：转换为 UTF-8 字节
byte[] text = StringToUtf8Bytes("Hello");
CreateLabel(hwnd, text, text.Length, ...);
```

## 🚀 扩展建议

### 1. 封装辅助类

```csharp
public class EmojiWindow : IDisposable
{
    private IntPtr _hwnd;
    
    public EmojiWindow(string title, int width, int height)
    {
        byte[] titleBytes = EmojiWindowNative.StringToUtf8Bytes(title);
        _hwnd = EmojiWindowNative.create_window_bytes(
            titleBytes, titleBytes.Length, width, height
        );
    }
    
    public void Dispose()
    {
        if (_hwnd != IntPtr.Zero)
        {
            EmojiWindowNative.destroy_window(_hwnd);
            _hwnd = IntPtr.Zero;
        }
    }
}
```

### 2. 控件封装

```csharp
public class EmojiLabel
{
    private IntPtr _hwnd;
    
    public string Text
    {
        get => GetText();
        set => SetText(value);
    }
    
    private void SetText(string text)
    {
        byte[] bytes = EmojiWindowNative.StringToUtf8Bytes(text);
        EmojiWindowNative.SetLabelText(_hwnd, bytes, bytes.Length);
    }
}
```

### 3. 事件封装

```csharp
public class EmojiButton
{

        Click?.Invoke(this, EventArgs.Empty);
    }
}
```
