# C# 示例代码

✅ **已完成！**

本目录包含 emoji_window.dll 的 C# 调用示例，支持 32位 和 64位。

## 📁 项目结构

```
Csharp/
├── EmojiWindowDemo_x86/          # 32位示例项目
├── EmojiWindowDemo_x64/          # 64位示例项目
├── EmojiWindowDemo.sln           # Visual Studio 解决方案
├── build.bat                     # 编译脚本
├── run.bat                       # 运行脚本
└── README.md                     # 详细文档
```

## 🚀 快速开始

### 方法 1：使用编译脚本（推荐）

```cmd
# 双击运行
build.bat

# 选择编译选项
[1] 编译 32位版本
[2] 编译 64位版本
[3] 编译全部版本
```

### 方法 2：使用 Visual Studio

1. 双击打开 `EmojiWindowDemo.sln`
2. 选择项目（x86 或 x64）
3. 按 F5 运行

### 方法 3：使用命令行

```cmd
# 32位
cd EmojiWindowDemo_x86
dotnet build -c Release

# 64位
cd EmojiWindowDemo_x64
dotnet build -c Release
```
