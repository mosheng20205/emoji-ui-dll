# 侧栏折叠菜单树形框升级版

在《侧栏折叠菜单树形框示例》基础上，左侧仍为 **侧栏树形框**，右侧为 **内容区**：点击不同菜单项时切换右侧显示（可用 **分组框** 或 **TabControl**，亦可混用）。

**可运行完整易语言源码**（与本说明同目录，复制到 IDE 后按 `DLL命令.md` 声明 DLL）：[`侧栏折叠菜单升级版_方案一分组框.md`](侧栏折叠菜单升级版_方案一分组框.md)（多分组框 + `显示分组框`）、[`侧栏折叠菜单升级版_方案二TabControl.md`](侧栏折叠菜单升级版_方案二TabControl.md)（`创建TabControl` + `切换到Tab`）。

---

## 布局示意

```
┌──────────────────────────────────────────────────────────────┐
│  主窗口（如 900×600）                                            │
├─────────────┬────────────────────────────────────────────────┤
│  树形框      │  右侧内容区（从 X≈340 起，宽≈550，高≈520）              │
│  宽 320      │  ┌──────────────────────────────────────────┐  │
│  左 10       │  │ 方案一：多个分组框叠放，只显示一个            │  │
│              │  │ 方案二：一个 TabControl，程序切换选中页        │  │
│              │  └──────────────────────────────────────────┘  │
└─────────────┴────────────────────────────────────────────────┘
```

- 树形框：`创建树形框 (主窗口, 10, 40, 320, 520, …)`（与原版一致）。
- 右侧起点：**X = 10 + 320 + 间距**，例如 **340**；**Y** 与树形框对齐，如 **40**；**宽度 = 客户区宽 − 右边距 − 起点 X**（固定窗口可先写死 **550**）。

---

## 交互要点

1. **树选中回调**（`设置树形框回调` 类型 **1** = 选中）里根据 **节点 ID** 决定右侧显示哪一块内容。
2. **节点 ID 与界面映射**：在 `添加根节点` / `添加子节点` 返回值保存到程序集变量（与原版相同），在回调里用 `.判断` / `.如果真` 分支切换。
3. **子节点点击**：若希望「订单管理 → 全部订单」也切换不同内容，同样在回调里判断 **节点ID**（包括子节点 ID）。

---

## 方案一：多个分组框叠放（推荐，直观）

思路：在**同一父窗口、同一矩形**上创建多个 **分组框**，默认全部创建后只 **显示一个**，其余 `显示分组框 (句柄, 假)`。选中菜单时 **隐藏当前**、**显示目标**。

**优点**：与树节点一一对应，不依赖 Tab 标题。  
**缺点**：控件较多；窗口缩放时需对每个分组框 `设置分组框位置`。

### DLL 要点（见 `DLL命令.md`）

- `创建分组框` — 父窗口为主窗口；标题为 UTF-8 字节集；颜色为 **十进制 ARGB**（勿写 `#RRGGBB`）。
- `显示分组框 (分组框句柄, 显示)` — 显示=真/假。
- 子控件（标签、编辑框等）父窗口设为 **分组框句柄**，并用 `添加子控件到分组框` 纳入分组框（按文档要求）。

### 程序集变量示例

```
.程序集变量 分组_欢迎, 整数型
.程序集变量 分组_订单, 整数型
.程序集变量 分组_通道, 整数型
.程序集变量 分组_财务, 整数型
.程序集变量 分组_API, 整数型
.程序集变量 分组_账号, 整数型
.程序集变量 当前右侧面板, 整数型
```

`当前右侧面板` 可存当前可见分组框句柄，切换时先 `显示分组框 (当前, 假)` 再 `显示分组框 (目标, 真)`。

### 切换子程序（逻辑示例）

```
.子程序 切换右侧内容, , , 根据树节点 ID 显示对应分组框
.参数 节点ID, 整数型

' 先全部隐藏（或只隐藏「当前」再显示目标，二选一）
显示分组框 (分组_欢迎, 假)
显示分组框 (分组_订单, 假)
…

.判断开始 (节点ID ＝ 节点_商户中心)
    显示分组框 (分组_欢迎, 真)
    当前右侧面板 ＝ 分组_欢迎
.判断 (节点ID ＝ 节点_订单管理)
    显示分组框 (分组_订单, 真)
    当前右侧面板 ＝ 分组_订单
' … 其它一级菜单与子节点分支 …
.默认
    显示分组框 (分组_欢迎, 真)
.判断结束
```

在 **回调_树选中** 末尾调用：`切换右侧内容 (节点ID)`。

## 方案二：单个 TabControl（右侧为多页）

思路：主窗口上 **创建一个 TabControl** 铺满右侧；`添加Tab页` 若干页（如「概览」「订单」「通道」…），树节点与 Tab 索引做 **显式映射**。若是易语言树选中回调，优先用“记下节点 ID → 下一拍再切换 Tab”的方式，避免在 DLL stdcall 回调栈里直接联动右侧 UI。

**优点**：自带页签，用户也可点 Tab 切换（若需禁止，可只用程序切换、隐藏 Tab 栏需查是否支持 `SetTab` 相关 API）。  
**缺点**：树与 Tab 的 **索引映射** 要维护；子节点多时 Tab 页数可能较多。

### DLL 要点

- `创建TabControl (主窗口, X, Y, 宽, 高)`
- `添加Tab页 (Tab句柄, 标题字节集指针, 长度, 0)` — 内容区传 **0** 由 DLL 自动创建内容窗口。
- `获取Tab内容窗口 (Tab句柄, 索引)` — 在对应页里 **创建标签、编辑框** 等（父窗口为该内容窗口句柄）。
- `切换到Tab (Tab句柄, 索引)` — 与树选中联动；易语言里建议放到时钟/延后子程序里执行。
- 窗口尺寸变化时：`设置TabControl位置` + `更新TabControl布局`（若使用缩放回调）。

### 安全写法示例

```
.子程序 回调_树选中, , , stdcall
.参数 节点ID, 整数型
.参数 上下文, 整数型

待切换树节点ID ＝ 节点ID
时钟_树选中延迟.禁止 ＝ 假

.子程序 _时钟_树选中延迟_周期事件
时钟_树选中延迟.禁止 ＝ 真
调用子程序 (&_按节点切换Tab, 待切换树节点ID, , )
```

不要在 `回调_树选中` 里再调用 `取树节点父节点` 推导 Tab。
如果日志出现 `GetNodeParent: enter hwnd=..., node_id=475211385` 这类大数字，通常就是把句柄或临时变量误当成节点 ID 传进去了。
如果编译时报“找不到指定的变量名称 `时钟_树选中延迟`”，说明你还没在窗口设计器里放这个时钟组件。
不想加组件时，也可以先直接在 `回调_树选中` 里调用 `调用子程序 (&_按节点切换Tab, 节点ID, , )`；若运行时还崩，再切回时钟延后方案。

## 与原版示例的衔接

1. 复制 `侧栏折叠菜单树形框示例.md` 中的 **创建主窗口、树形框、侧栏模式、构建树、回调注册**。
2. 在 `设置树形框回调 (…, 1, &回调_树选中)` 所指向的子程序中，分组框方案可直接切内容；TabControl 方案建议只记录 `节点ID`，再由时钟/延后子程序调用 **`切换右侧内容 (节点ID)`** 或 **`切换到Tab`**。
3. 启动时：创建右侧分组框或 TabControl → 再 **默认选中** 某一树节点并 **同步显示** 第一屏（与 `设置选中树节点` 一致）。

## 常量与颜色

- ARGB 请用 **十进制** 或程序内 `.常量`，勿在参数位置写 `#FF303133`（易语言 `#` 为常量名，易报错 55）。
- 可参考 `易语言代码文档\常量表.md`。

---

## 可选：窗口缩放

若使用 `设置窗口大小改变回调`，在回调里用 **新宽高** 重新计算右侧 `SetTabControlBounds` / `SetGroupBoxBounds`，并 `更新TabControl布局`。

## 最小可运行骨架（方案一 + 选中回调）

下列为 **结构示意**；`创建分组框` 参数较多，请按 `DLL命令.md` 中实际参数顺序补全（标题指针、边框色、背景色、字体指针可传 0 表示默认，以文档为准）。

```
.程序集变量 主窗口句柄, 整数型
.程序集变量 树形框句柄, 整数型
.程序集变量 分组_A, 整数型
.程序集变量 分组_B, 整数型
' … 节点_xxx 与原版相同 …

.子程序 _启动子程序, 整数型
' … 创建主窗口、创建树形框、构建树、设置回调 …

' 右侧两个分组框，同一区域 340,40,540,500
分组_A ＝ 创建分组框 (主窗口句柄, 340, 40, 540, 500, …, “面板A” UTF-8…, …)
分组_B ＝ 创建分组框 (主窗口句柄, 340, 40, 540, 500, …, “面板B” UTF-8…, …)
显示分组框 (分组_A, 真)
显示分组框 (分组_B, 假)

设置树形框回调 (树形框句柄, 1, &回调_树选中)
' …
返回 (0)

.子程序 回调_树选中, , , stdcall
.参数 节点ID, 整数型
.参数 上下文, 整数型

.如果 (节点ID ＝ 节点_商户中心)
    显示分组框 (分组_A, 真)
    显示分组框 (分组_B, 假)
.否则
    显示分组框 (分组_A, 假)
    显示分组框 (分组_B, 真)
.如果结束
```

按实际节点数量扩展 **分组框** 与 **分支** 即可与截图中多菜单一致。
