# 所有控件属性操作综合示例

本示例展示如何使用 emoji_window.dll 的所有控件属性 API，包括按钮、窗口、标签、复选框、单选按钮、分组框、TabControl、编辑框和进度条的属性获取与设置。

## 功能说明

本综合示例演示以下所有控件的属性操作：
1. 按钮属性（文本、Emoji、位置、背景色、可视、启用）
2. 窗口属性（标题、位置、可视、标题栏颜色）
3. 标签属性（文本、字体、颜色、位置、对齐、启用、可视）
4. 复选框属性（文本、字体、颜色）
5. 单选按钮属性（文本、字体、颜色）
6. 分组框属性（标题、位置、可视、启用、颜色）
7. TabControl 属性（Tab 标题、位置、可视、启用）
8. 编辑框属性（字体、颜色、位置、对齐、启用、可视）
9. 进度条属性（颜色、位置、启用、可视、显示文本）

## 使用说明

### 窗口布局

本综合示例窗口分为左右两个区域：

**左侧区域（X: 10~530）**
- 顶部：结果显示标签（实时显示操作结果）
- 演示控件行：3个演示按钮、演示标签、演示复选框、演示单选按钮、演示分组框
- 演示控件行：演示编辑框
- 操作按钮区：按钮属性、窗口属性、标签属性操作按钮

**右侧区域（X: 540~1090）**
- 演示 TabControl（2个Tab页）
- 演示进度条
- 操作按钮区：复选框/单选按钮、分组框、TabControl、编辑框、进度条属性操作按钮

### 两次调用模式

获取文本类属性时使用两次调用模式：

```
' 第一次：传 0 获取所需字节长度
文本长度 = 获取按钮文本 (按钮ID, 0, 0)

' 第二次：分配缓冲区后获取内容
文本缓冲区 = 取空白字节集 (文本长度)
获取按钮文本 (按钮ID, 取变量数据地址 (文本缓冲区), 文本长度)
```

### UTF-8 字节集拼接规则

DLL 返回的缓冲区本身是 UTF-8 字节集，**不能用 `到文本()` 转换**，必须全程字节集拼接：

```
' 正确：字节集拼接
结果字节集 = 编码_Ansi到Utf8 ("文本：") + 文本缓冲区 + { 10 }
设置标签文本 (标签_结果, 取变量数据地址 (结果字节集), 取字节集长度 (结果字节集))
```

### 颜色格式

| 函数 | 颜色格式 | 说明 |
|------|---------|------|
| 按钮背景色 | ARGB（32位） | `到ARGB(A, R, G, B)` |
| 标签/编辑框/进度条颜色 | ARGB（32位） | `到ARGB(A, R, G, B)` |
| 窗口标题栏颜色 | RGB（24位） | `到RGB(R, G, B)` |

### 传址参数

`获取按钮位置`、`获取窗口位置`、`获取标签位置` 等函数的 X、Y、宽度、高度参数需要传址：

```
.局部变量 X, 整数型
.局部变量 Y, 整数型
.局部变量 宽度, 整数型
.局部变量 高度, 整数型
获取按钮位置 (按钮ID, X, Y, 宽度, 高度)
```
