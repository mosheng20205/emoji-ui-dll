' ========== 按钮启用/禁用功能使用说明 ==========
' 
' 新增了两个 DLL 命令用于控制按钮的启用/禁用状态：
' 
' 1. 启用按钮 (父窗口句柄, 按钮ID, 启用状态)
'    - 父窗口句柄: 按钮所在的父窗口句柄
'    - 按钮ID: 要控制的按钮ID
'    - 启用状态: 真=启用，假=禁用
' 
' 2. 禁用按钮 (父窗口句柄, 按钮ID)
'    - 这是 启用按钮 的快捷方式，等同于 启用按钮(父窗口句柄, 按钮ID, 假)
' 
' 使用示例：
' 
' ' 禁用"启用分组框1"按钮
' 启用按钮 (窗口句柄, 按钮2, 假)
' 
' ' 或者使用快捷方式
' 禁用按钮 (窗口句柄, 按钮2)
' 
' ' 重新启用按钮
' 启用按钮 (窗口句柄, 按钮2, 真)
' 
' 注意事项：
' - 禁用的按钮会显示为灰色，无法点击
' - 禁用的按钮不会触发悬停效果
' - 分组框被禁用时，分组框内的所有按钮也会自动禁用
' - 分组框被启用时，分组框内的所有按钮也会自动启用
