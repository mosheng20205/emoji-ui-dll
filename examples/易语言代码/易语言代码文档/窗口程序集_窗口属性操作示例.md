# 窗口属性操作示例

本示例展示如何使用emoji_window.dll的窗口属性API动态获取和修改窗口的各种属性。

## 功能说明

本示例演示以下窗口属性操作：
1. 获取窗口标题
2. 获取和设置窗口位置和大小
3. 获取和设置窗口可视状态
4. 获取窗口标题栏颜色

## 注意事项

1. **两次调用模式**：获取窗口标题时需要使用两次调用模式
   - 第一次传入buffer=0获取所需长度
   - 第二次传入实际缓冲区获取内容

2. **UTF-8编码**：窗口标题使用UTF-8编码，需要使用`到字节集()`转换

3. **RGB颜色格式**：标题栏颜色使用RGB格式（24位整数）
   - Red（红色）：最高8位
   - Green（绿色）：中间8位
   - Blue（蓝色）：最低8位

4. **传址参数**：`获取窗口位置`的X、Y、宽度、高度参数需要使用传址方式传递

5. **立即生效**：所有Set函数修改后会立即生效，无需手动刷新

6. **错误处理**：
   - `获取窗口标题`返回-1表示失败
   - `获取窗口可视状态`返回-1表示失败
   - `获取窗口标题栏颜色`返回-1表示失败

7. **窗口隐藏**：当窗口被隐藏时，所有子控件也会隐藏
