# Emoji Window DLL（C++）

基于 **Direct2D / DirectWrite** 的 Windows 原生 UI 控件 DLL：彩色 **Emoji** 与 Unicode 文本显示、多套主题（亮/暗）、布局与表格等能力，面向 **易语言**、也可配合 **C# / Python** 等通过 P/Invoke 或 ctypes 调用。仓库内含源码、主题 JSON、以及各语言示例。
